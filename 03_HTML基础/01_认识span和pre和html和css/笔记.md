## 基础

### HTML5中的语义化

1. 每个元素都用具体的含义


2. 所有元素与展示效果无关。是默认的CSS在起效果。
   元素展示在页面中的效果，应该由CSS决定。 
   因为浏览器有一些自带的CSS样式，所以每个元素都有一定的样式。


**选择什么元素，取决于内容的含义，而不是显示中的效果**

### 实体字符
